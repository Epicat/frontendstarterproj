          About this file

          What it is
          It is a collection of basic unformatted commonly used html tags
          It is a litmus test of how well the design has been implemented
          It is a work in progress. Crappy coverage? Add, change, adjust, tweak!
          What it isn't
          It is not a library of components for the site
          It is not a mood board for the client to sign off on
          A list of all/allowed html tags
          Something that should end up in in the production environment
          What it's purpose is
          It is meant to be a showcase of the sites general appearance
          Once the basic design has been set the contents of this file should look good at any given time
          It should expose lack of coverage of the design implementation
